YouTube Smart Transcript - Test Page

This page helps you test your YouTube Smart Transcript extension. Follow the instructions below to verify that your extension is working correctly.

Test Cases

Test Case 1: Non-Coding Video

Visit a YouTube video that is not related to coding (e.g., a music video, documentary, etc.).

  1. Go to a non-coding YouTube video (e.g., this example)
  2. Click on the YouTube Smart Transcript extension icon
  3. Click the "Get Transcript" button
  4. Expected result: You should see important points from the video transcript

Test Case 2: Coding Video

Visit a YouTube video that is related to coding or programming.

  1. Go to a coding-related YouTube video (e.g., this JavaScript tutorial)
  2. Click on the YouTube Smart Transcript extension icon
  3. Click the "Get Transcript" button
  4. Expected result: You should be redirected to YouTube's home page

Troubleshooting

  • Extension icon not appearing: Make sure the extension is properly installed. Go to chrome://extensions/ and check if it's enabled.
  • Transcript not loading: Some videos may not have transcripts available. Try with a different video.
  • Incorrect detection: The extension uses simple keyword matching to detect coding videos. It may not always be accurate.
  • Extension not working at all: Check the browser console for any error messages.
